# peer-papel-tesoura
Um jogo online de Pedra-Papel-Tesoura para jogar com os amigos que
possuem IPv6.

Esse é um projeto de estudo sobre como fazer um netcode jogos online,
utilizando do conceito de peer-to-peer.

O IPv6 é obrigatório[^1] por ser o principal protocolo que permite
comunicação direta na internet moderna, sem a necessidade de
servidores internediários/relay.

[^1]: É possível sim implementar servidores relay para clientes que
possuem apenas acesso a IPv4, porém não é o foco desse estudo.
